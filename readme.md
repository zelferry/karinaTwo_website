# website da karinaTwo

## arquivo .env:
```env
TOKEN="" #token do bot
CLIENT_SECRET="" #secret xo cliente do bot
MONGOOSE="" #mongoose url
CLIENT_ID="" #id do bot
SECRET="" #secret(pode ser qualquer coisa)
```

## iniciar o site:

```node
node index.js
```
ou:
```npm
npm start
```

site feito em EXPRESS e EJS
